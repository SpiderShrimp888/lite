# 唯一编号生成机制

| 索引  | 含义                                | 位数 |
| ----- | ----------------------------------- | ---- |
| 1~8   | 生成ID的时间戳，16进制              | 8    |
| 9~14  | 计算机名字hash值，16进制            | 6    |
| 15~22 | 进程编号，16进制                    | 8    |
| 23~30 | 自增数值，范围1~max(uint32)，16进制 | 8    |
